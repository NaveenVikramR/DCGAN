# DCGAN From-Scratch Reproduction

Reproduction of [Unsupervised Representation Learning with Deep Convolutional GANs](https://arxiv.org/abs/1511.06434) (Radford, Metz, Chintala, 2015), trained on the Anime Face Dataset, evaluated with FID, and compared against a vanilla fully-connected GAN baseline.

Status: Phase 0 (project skeleton) complete. See `DCGAN_Build_Spec.md` for the full phase-by-phase plan.
